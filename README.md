## Patterns Covered 

- **Hash Map / Set** — Trading space for O(n) time; frequency counting
- **Sliding Window** — Variable-size window for substring/subarray problems
- **Two Pointers** — Converging from both ends; sorted array optimisation 
- **Sorting + Sweep** — Sort first to simplify; interval merging
- **Stack** — LIFO for matching/nesting problems (parentheses, monotonic)
- **Linked List** — Pointer manipulation; fast/slow runner technique
- **Prefix Sums** — Pre-computation for range queries
- **Basic DP** — Memoisation for overlapping subproblems
